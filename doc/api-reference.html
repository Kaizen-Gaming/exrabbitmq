    ExRabbitMQ v1.1.0 API Reference

    Modules

    ExRabbitMQ

    A project providing the following abstractions

    ExRabbitMQ.AST.Common

    AST holding module for the consumer and producer behaviours

    ExRabbitMQ.AST.Consumer.GenServer

    AST holding module for the consumer behaviour when the surrounding consumer is a GenServer

    ExRabbitMQ.AST.Consumer.GenStage

    AST holding module for the consumer behaviour when the surrounding consumer is a GenStage

    ExRabbitMQ.AST.Producer.GenServer

    AST holding module for the producer behaviour when the surrounding producer is a GenServer

    ExRabbitMQ.Connection

    A GenServer implementing a long running connection to a RabbitMQ server, with embedded PubSub using :pg2 and :ets

    ExRabbitMQ.ConnectionConfig

    A stucture holding the necessary information about a connection to a RabbitMQ node

    ExRabbitMQ.ConnectionSupervisor

    A supervisor using the :simple_one_for_one strategy to serve as a template for spawning new RabbitMQ connection (module ExRabbitMQ.Connection) processes

    ExRabbitMQ.Constants

    A module holding constants (e.g., process dictionay keys) that are used in multiple sites

    ExRabbitMQ.Consumer

    A behaviour module that abstracts away the handling of RabbitMQ connections and channels

    ExRabbitMQ.Consumer.QueueConfig

    A stucture holding the necessary information about a queue that is to be consumed

    ExRabbitMQ.Producer

    A behaviour module that abstracts away the handling of RabbitMQ connections and channels
